# 📱 Student Social Media and Mental Health Impact

## 📌 Project Overview

This project analyzes the impact of social media usage on students’ mental health and well-being. Using **Tableau, Excel, and data analytics**, the project explores how social media habits relate to factors such as anxiety, sleep quality, academic performance, relationship status, and overall emotional health.

The goal of this project is to present student behavior and mental health trends through an interactive dashboard that helps identify patterns, risks, and insights related to social media usage.

## 🔗 Live Dashboard

View the interactive Tableau dashboard here:

[Student Social Media and Mental Health Impact Dashboard](https://public.tableau.com/app/profile/preetii.birajdar/viz/StudentSocialMediaAndMentalHealthImpact/Dashboard1)

## 🎯 Business Problem

Social media is a major part of student life, but excessive usage may affect mental health, sleep, academic focus, and emotional well-being. Without proper analysis, it can be difficult to understand:

- How much time students spend on social media
- Which groups may be more affected by heavy usage
- Whether social media usage is linked with anxiety, depression, or sleep issues
- How student lifestyle factors relate to mental health indicators
- Which patterns require more awareness and support

This project addresses the need to better understand the relationship between social media behavior and student mental health through data visualization.

## 🎯 Objectives

- Analyze student social media usage patterns
- Explore the relationship between screen time and mental health indicators
- Compare mental health trends across gender, age, academic level, and relationship status
- Identify possible links between social media usage, sleep, and academic performance
- Build an interactive Tableau dashboard for easy exploration
- Communicate insights in a clear and visual format

## 🛠️ Tools and Technologies Used

- **Tableau Public** – Interactive dashboard creation and data visualization
- **Microsoft Excel** – Data cleaning, formatting, and preparation
- **Data Analytics** – Exploratory analysis, trend identification, and insight generation
- **GitHub** – Project documentation and portfolio presentation

## 📊 Dashboard Features

The Tableau dashboard is designed to highlight key student behavior and mental health metrics such as:

- Average daily social media usage
- Social media usage by student groups
- Mental health indicators such as anxiety, depression, and stress
- Sleep pattern comparison
- Academic impact analysis
- Relationship between social media habits and emotional well-being
- Interactive filters for deeper analysis

Dashboard Action Used

A Filter Action was added to make the dashboard interactive.

When a user clicks on a country in the world map, the other charts and tables automatically update to show data only for the selected country.

Action Settings:

- Source Sheet: World Map
- Target Sheets: Purpose of Use, Sleep Analysis, Platform Analysis, Student Details
- Run Action On: Select
- Clearing Selection: Show All Values
- Target Filter: Country → Country

This action allows users to explore country-wise differences in student social media usage, sleep patterns, mental health scores, addiction levels, and academic performance.

## 📈 Key Insights

Some of the major insights explored in this project include:

- Students who spend more time on social media may report higher levels of mental health concerns.
- Sleep quality and duration may vary based on social media usage habits.
- Social media usage patterns can differ across gender, age group, academic level, and relationship status.
- Heavy social media use may be associated with reduced academic focus and emotional well-being.
- Interactive visualization helps identify student groups that may need more awareness or support.

## 💡 Recommendations

Based on the analysis, the following recommendations can support healthier social media usage among students:

- Encourage students to monitor and limit excessive daily social media usage.
- Promote awareness about the possible connection between screen time, sleep, and mental health.
- Support students with high anxiety, stress, or depression indicators through wellness programs.
- Encourage balanced digital habits, especially among students with high usage patterns.
- Use data-driven insights to guide mental health awareness campaigns in schools and universities.

## 🧠 Why This Project Matters

Social media can help students communicate, learn, and stay connected, but excessive usage may also contribute to stress, anxiety, sleep disruption, and reduced academic focus.

This project demonstrates how data visualization can be used to understand student behavior, identify potential well-being risks, and support conversations around responsible social media use and mental health awareness.
