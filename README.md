# Student Social Media and Mental Health Impact Dashboard

## 📌 Project Overview

This project explores the relationship between students’ social media usage and mental health indicators through an interactive Tableau dashboard. The dashboard is designed to help users understand how factors such as daily social media usage, sleep patterns, academic level, relationship status, and emotional well-being may be connected.

The goal of this project is to present student mental health and social media behavior data in a clear, visual, and interactive way so that trends and patterns can be identified more easily.

## 🔗 Live Dashboard

View the interactive Tableau dashboard here:

[Student Social Media and Mental Health Impact Dashboard](https://public.tableau.com/app/profile/preetii.birajdar/viz/StudentSocialMediaAndMentalHealthImpact/Dashboard1)

## 🎯 Objectives

- Analyze students’ social media usage patterns
- Identify possible relationships between social media use and mental health
- Explore how usage varies by gender, age, academic level, and relationship status
- Visualize mental health indicators such as anxiety, depression, sleep issues, and self-esteem
- Build an interactive dashboard for easy data exploration

## 📊 Dashboard Features

The Tableau dashboard includes visual insights such as:

- Average daily social media usage among students
- Mental health scores based on student responses
- Comparison of social media usage across different student groups
- Relationship between screen time and sleep patterns
- Impact of social media on academic and emotional well-being
- Interactive filters for deeper analysis

## 🛠️ Tools and Technologies Used

- **Tableau Public** – Dashboard creation and data visualization
- **Microsoft Excel / CSV** – Data cleaning and preparation
- **Data Analytics** – Exploratory analysis and insight generation
